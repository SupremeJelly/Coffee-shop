# Coffee-shop

- Dự án quản lý quán cà phê, được xây dựng bằng C#.
- Thiết kế theo kiến trúc 3 lớp.
- Sử dụng 2 kiểu cơ sở dữ liệu khác nhau là SQLServer và MongoDB.
- Phiên bản sử dụng SQL server có hỗ trợ Backup, Restore và Encode dữ liệu.
- Phát triển theo mô hình ứng dụng desktop .NET (Windows Forms).
- Hỗ trợ các nghiệp vụ chính như bán hàng, đặt hàng, nhập hàng và báo cáo.
- Có tổ chức nhiều màn hình/chức năng theo từng module để dễ sử dụng và quản lý.
